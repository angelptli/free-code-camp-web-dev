Introduction

Python is known as one of the most human readable programming languages. It allows us to write and understand code more quickly because we don't have to deal with many declarations that are usually required in low-level programming languages. That said, Python is a high-level language.

Prerequisite

This guide assumes you:

  • Are interested in the Python programming language
  • Have basic familiarity with any programming language
  • Follow and use the example code to gain deeper knowledge on Python
Hello World

Get started with the famous starter of every programming language:

print("Hello World")

Variables

Name variables to represent the object you want to work with. For example, if your making a money counting program you might use variables called cash, account, and total.

Variables must start with either:

  • a character from a-z
  • an underscore (_)
  • a dollar sign ($)

Subsequent characters can be any of the above listed or a digit from 0-9. Variables in are case sensitive, although normally variables in Python are lower case when using a character from a-z.

Declaring Variables

Here is the conventional way to declare a variable in Python:

name = value

The value can be anything from a number, string, character, list, set, dict, and so on.

Variable Scope

The scope of a variable is in the function it is declared in. It is convention to declare variables in functions. There are other kinds of variables. A function can be called with arguments and accept those values as parameter variables, which allow those to be within the scope of that function. 

def function_name(param1, param2):
    var_name = value
    print(param1, param2, var_name)

This function has three variables in scope

Global Variables

Variables declared outside of a function or class are known as global varaibles. Their scope is the whole program and its value is accessible everywhere. Global variables are declared in all upper case to distinguish them:

GLOBAL_VAR_NAME = value

Global variable value are usually not meant to be altered and should only be accessed.

if...else Statement

You can define conditions that must be met before statements are run. Here is the basic syntax of an if-else statement: 

if (condition)
    statement(s)
else
    statement(s)

The else statement is optional.

You can also define compound conditions using if, elif, and else statements in combination: 

if (condition)
    statement(s)
elif (condition)
    statement(s)
...
else
    statement(s)

You can only define one if- and else-statement, but you can define unlimit elif statements. Note: elif stands for else if.
